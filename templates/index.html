📌 O que é Python?

🔍 Conceito

Python é uma linguagem de programação de alto nível, interpretada, com sintaxe simples e legível. Foi criada por Guido van Rossum em 1991 e atualmente é uma das linguagens mais populares do mundo, utilizada por empresas como Google, Netflix, NASA e Spotify.

  • Inteligência Artificial - Machine Learning com TensorFlow e PyTorch
  • Automação - Scripts para tarefas repetitivas
  • Cybersegurança - Análise de vulnerabilidades
  • IoT - Programação para Raspberry Pi
  • Desenvolvimento de Jogos - Com Pygame

📦 O que é POO (Programação Orientada a Objetos)?

🔍 Conceito

Programação Orientada a Objetos (POO) é um paradigma de programação que organiza o código em torno de "objetos" do mundo real. Esses objetos possuem atributos (dados/características) e métodos (comportamentos/ações).

A POO é amplamente utilizada em linguagens como Python, Java, C++, C#, JavaScript e PHP.

🏗️ Conceitos fundamentais

  • Classe - Molde ou blueprint para criar objetos. Define atributos e métodos.
  • Objeto - Instância concreta de uma classe. Ex: um carro específico da classe Veículo.
  • Atributo - Características do objeto (cor, tamanho, preço, modelo).
  • Método - Ações que o objeto pode executar (acelerar, frear, ligar).

🎯 Pilares da POO

  • Herança - Uma classe filha herda atributos e métodos da classe pai, promovendo reutilização.
  • Encapsulamento - Protege dados internos do objeto, controlando o acesso externo.
  • Polimorfismo - Um mesmo método pode ter comportamentos diferentes em classes distintas.
  • Abstração - Esconde detalhes complexos de implementação, mostrando apenas o essencial.

✅ Vantagens de utilizar POO

  • Reutilização de código - Herança evita repetição e segue o princípio DRY (Don't Repeat Yourself)
  • Organização - Código mais limpo, estruturado e de fácil compreensão
  • Manutenção facilitada - Mudanças são localizadas e não afetam todo o sistema
  • Segurança - Encapsulamento protege dados sensíveis de acesso indevido
  • Modelagem real - Representa melhor objetos e situações do mundo real
  • Escalabilidade - Projetos grandes e complexos ficam gerenciáveis
  • Produtividade em equipe - Vários desenvolvedores podem trabalhar em classes diferentes simultaneamente
  • Baixo acoplamento - Classes são independentes, facilitando testes e reuso

📝 Exemplo prático em Python

class Veiculo:
    def __init__(self, marca, modelo):
        self.marca = marca      # atributo
        self.modelo = modelo    # atributo
    
    def ligar(self):
        return f"{self.marca} {self.modelo} ligado!"  # método

# Criando um objeto (instância da classe)
meu_carro = Veiculo("Toyota", "Corolla")
print(meu_carro.ligar())  # Saída: Toyota Corolla ligado!

⬆️ Este exemplo mostra uma classe Veiculo, com atributos (marca, modelo) e método (ligar), e um objeto meu_carro criado a partir dela.

📊 Comparação: POO vs Programação Estruturada

📋 Programação Estruturada

  • Organizada em funções/procedimentos
  • Dados separados das funções
  • Menos reutilização de código
  • Mais difícil para projetos grandes
  • Exemplos: C, Pascal

🎯 Programação Orientada a Objetos

  • Organizada em objetos com dados e métodos
  • Dados e funções juntos (encapsulados)
  • Alta reutilização via herança
  • Ideal para projetos grandes e complexos
  • Exemplos: Python, Java, C++
